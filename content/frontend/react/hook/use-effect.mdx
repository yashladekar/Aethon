# useEffect

React's synchronization hook.

## Basic Example

```tsx
useEffect(() => {
  console.log("Mounted");
}, []);
```

## Cleanup

```tsx
useEffect(() => {
  return () => {
    console.log("Cleanup");
  };
}, []);
```
